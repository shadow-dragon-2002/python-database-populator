1. Testing Access Denied Error:
✗ Failed to connect to database: 2003 (HY000): Can't connect to MySQL server on 'localhost:3306' (111)

🔧 Troubleshooting Guide:
❌ Connection Error:
   • Check if database server is running on localhost:3306
   • Verify the host IP address/hostname is correct
   • Check firewall settings and network connectivity
   • For local testing, try host='localhost' or '127.0.0.1'

📋 Current Configuration:
   • Database Type: mysql
   • Host: localhost
   • Port: 3306
   • Database: test_db
   • Username: wrong_user
   • Password: **************

💡 Quick Solutions:
   • For local MySQL: Use host='localhost', user='root'
   • For remote MySQL: Ensure user has remote access privileges
   • Try: mysql -h {host} -u {username} -p {database}
